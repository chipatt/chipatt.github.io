Qu'est-ce que .htaccess ?

Fichier de configuration Apache placé à la racine du site. Contrôle redirections, cache, compression, sécurité.

Directives Courantes

Directive Fonction
RewriteEngine Active le module rewrite
RewriteRule Définit une règle de réécriture
RedirectMatch Redirige vers nouvelle URL
ExpiresActive Contrôle la mise en cache
mod_gzip Compresse le contenu

FAQ .htaccess

.htaccess fonctionne sur tous les hébergements ?

Non. Seulement sur serveurs Apache. Pas sur Nginx ou cloud modernes.

Où placer le fichier .htaccess ?

À la racine du site web (public_html ou www)

Je peux copier/coller le contenu généré ?

Oui, créez simplement un fichier texte nommé ".htaccess"

Pourquoi mon .htaccess ne fonctionne pas ?

Erreur de syntaxe, serveur non Apache, modules non activés.
