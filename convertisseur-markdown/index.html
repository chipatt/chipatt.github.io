FAQ Markdown

Pourquoi utiliser Markdown ?

Markdown est simple, lisible, et peut être converti en HTML, PDF, ou d'autres formats.

Qui utilise Markdown ?

Développeurs, rédacteurs, chercheurs. Utilisé dans GitHub, Reddit, Discord, etc.

Y a-t-il différentes versions ?

Oui : CommonMark, GFM (GitHub Flavored Markdown), MultiMarkdown, etc.

Puis-je inclure du HTML dans Markdown ?

Oui, la plupart des convertisseurs acceptent du HTML brut.
