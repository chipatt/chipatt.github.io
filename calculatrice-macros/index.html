Comprendre les macronutriments

Les trois macros

Protéines (4 cal/g): Construisent et réparent muscles. Créent hormones et enzymes. Essentielles pour santé. Glucides (4 cal/g): Source d'énergie principale. Carburant pour cerveau et exercice. Fibres essentielles.

Lipides (9 cal/g): Plus calorique mais essentiel. Supportent hormones, cerveau, absorption vitamine. Graisses saines sont vitales.

Splits par objectif

Perte de poids: Haute protéine (35-40%) pour préserver muscle. Moins glucides (30-35%). Lipides modérés (25-30%). Maintien: Équilibré. 30% protéine, 40% glucides, 30% lipides.

Gain musculaire: Protéine très haute (35-40%). Glucides suffisants (45-50%) pour énergie. Lipides (20-25%). Performance: Très haute glucides (50-60%) pour énergie. Protéine modérée (20-25%). Lipides (20-25%).

Ajuster vos macros

Commencez avec le split calculé. Suivez comment vous vous sentez après 2 semaines. Trop fatigué? Augmentez glucides. Toujours affamé? Augmentez protéine. Les macros parfaites sont celles que vous pouvez suivre.

Questions Fréquentes

Dois-je compter les macros exactement?

Non, approximativement suffit. Être très proche est assez bon. L'important est la cohérence et atteindre environ votre objectif.

Les sources de macros importent-elles?

Oui! Les macros sont quantité, mais sources affectent santé. Préférez sources entières: œufs vs poudre, riz brun vs sucre blanc.

Dois-je sauter les graisses pour maigrir?

Non, graisses saines sont essentielles. Elles aident hormone, absorption vitamine, satiété. Limiter trop = problèmes.

Puis-je changer les macros si résultats pas bons?

Oui, mais donnez 2-3 semaines pour adapter. Changements trop fréquents = pas de données. Ajustez légèrement, pas radicalement.
