Questions Fréquemment Posées

Puis-je utiliser plusieurs minuteurs en même temps? +
Oui! Vous pouvez ajouter jusqu'à 5 minuteurs simultanés. Chacun fonctionne indépendamment et émet une alerte quand le temps est écoulé.
Le minuteur continue-t-il si je ferme l'onglet? +
Le minuteur s'arrête si vous fermez l'onglet. Gardez la page ouverte ou utilisez les minuteurs de votre téléphone pour une durée prolongée.
Y a-t-il une alerte sonore? +
Oui, une alerte sonore retentit quand le temps est écoulé. Assurez-vous que votre volume est activé.
