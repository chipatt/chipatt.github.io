Checklist Responsive

  • ✓ Viewport meta tag présent
  • ✓ Texte lisible (14px minimum)
  • ✓ Boutons cliquables (48x48px minimum)
  • ✓ Images responsive (max-width: 100%)
  • ✓ Pas de scroll horizontal
  • ✓ Touch-friendly sur mobile
  • ✓ Fast loading (<3s)

FAQ Responsive

Mobile-first ou desktop-first ?

Mobile-first est mieux pour SEO 2026. Commencez petit, montez.

Combien de breakpoints ?

3-4 suffisent : mobile, tablet, desktop, widescreen.

Flexbox ou Grid pour responsive ?

Les deux ! Flexbox pour menus, Grid pour layouts.

Comment tester vraiment ?

Vrais appareils ou DevTools Chrome. Cet outil est pour aperçu.
