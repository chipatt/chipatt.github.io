Questions Fréquemment Posées

Puis-je sauvegarder ma liste? +
Utilisez Ctrl+P pour imprimer votre liste. Vous pouvez aussi copier/coller le texte dans un document ou une note.
Comment cocher les articles? +
Cliquez sur les cases à cocher pour marquer les articles comme achetés. Les articles sont rayés visuellement.
Puis-je partager ma liste? +
Copiez la liste (Ctrl+A puis Ctrl+C) et collez-la dans un e-mail ou un message pour la partager avec quelqu'un d'autre.
