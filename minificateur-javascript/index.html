FAQ JavaScript

La minification casse-t-elle mon code ?

Non, si le code était correct avant. Testez toujours après.

Combien de taille j'économise ?

30-50% généralement, plus si beaucoup de commentaires.

Faut-il faire map de source (source map) ?

Oui, en production pour déboguer. Générez un fichier .map.

Outil recommandé pour production ?

Terser, UglifyJS, ou webpack. Pas ce simple outil.
