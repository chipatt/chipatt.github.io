Règles Générales

Fromages doux: Vins blancs légers et fruités. Fromages forts: Vins rouges robustes ou desserts. Fromages piquants: Vins acides (Sancerre, Riesling). Fromages crémeux: Bourgognes ou Chardonnays.

Questions Fréquemment Posées

Pourquoi accorder le vin avec le fromage? +
Les bons accords créent une harmonie gustative. Le vin complète ou contraste avec les saveurs du fromage.
Quelle est la meilleure température de service? +
Les vins blancs légers: 8-10°C. Les vins blancs ronds: 10-12°C. Les vins rouges: 14-18°C selon le style.
Puis-je aérer le vin avant de le servir? +
Oui, aérer permet aux arômes de se développer. Les vins rouges bénéficient généralement de 15-30 minutes d'aération.
