FAQ Minification CSS

Le CSS minifié fonctionne-t-il pareil ?

Absolument. La minification n'affecte que la présentation, pas la fonctionnalité.

Combien de taille puis-je économiser ?

En moyenne 40-50% selon le CSS. Plus de commentaires = plus d'économies.

Dois-je minifier mon CSS en développement ?

Non, gardez le CSS formaté en dev. Minifiez lors du déploiement.

Y a-t-il des risques ?

Non, tant que vous utilisiez du CSS valide. Testez après minification.
