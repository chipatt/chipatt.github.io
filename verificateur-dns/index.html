Vérificateur DNS en Ligne

Tout ce que vous devez savoir sur le DNS et comment le configurer correctement.

Qu'est-ce que le DNS ?

DNS (Domain Name System) est le système qui traduit les noms de domaine (www.example.com) en adresses IP (192.168.1.1). C'est comme un annuaire téléphonique pour Internet.

Types d'Enregistrements DNS

Type Fonction Exemple
A Mappe domaine vers IPv4 example.com → 192.168.1.1
AAAA Mappe domaine vers IPv6 example.com → 2001:db8::1
MX Routeur pour emails mail.example.com
CNAME Alias de domaine www → example.com
TXT Données texte SPF, DKIM, verifications
NS Serveurs de noms ns1.provider.com

Processus de Résolution DNS

  1. Vous entrez "google.com" dans le navigateur
  2. Votre navigateur demande au serveur DNS du FAI
  3. Le serveur cherche l'IP associée au domaine
  4. L'IP est retournée au navigateur
  5. Le navigateur se connecte à cette IP
  6. Le site web charge

Problèmes DNS Courants

  • DNS qui ne répond pas - Site inaccessible
  • Mauvaise IP - Vous dirigez vers le mauvais serveur
  • Propagation DNS - Nouveaux enregistrements peuvent prendre 24-48h
  • Cache DNS - Votre PC garde en mémoire l'ancienne IP

Comment Vérifier Votre DNS

Outils utiles :

  • ping - Tester la connectivité
  • nslookup - Chercher enregistrements DNS
  • dig - Version avancée de nslookup
  • tracert/traceroute - Tracer le chemin réseau
  • whois - Info sur propriétaire domaine

FAQ DNS

Combien de temps pour qu'un changement DNS prenne effet ?

Théoriquement 24-48h. En pratique souvent 1-2h. Vérifiez TTL (Time To Live).

Comment purger le cache DNS ?

Windows : ipconfig /flushdns | Mac : sudo dscacheutil -flushcache

DNS gratuit ou payant ?

Gratuit suffit généralement. Cloudflare (1.1.1.1) est populaire et gratuit.

Dois-je changer mes serveurs DNS ?

Optionnel. Changez pour vitesse, sécurité, ou contrôle parental.
